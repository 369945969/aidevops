# 知识库
团队知识资产的集中管理页面,自动沉淀每次迭代的需求、架构设计、代码模式、问题解决方案,提供强大的搜索和筛选功能,支持AI和团队成员复用历史经验。

Layout Hierarchy:
- Header (Full-width):
  - Top Navigation Bar
- Content Container (Positioned below the header):
  - Left Sidebar
  - Main Content Area

## Top Navigation Bar
- 产品Logo和名称
- 导航菜单:工作流看板、需求管理、知识库(当前选中)、系统设置
- 全局搜索框
- 新建需求按钮
- 通知中心图标
- 用户头像和下拉菜单

## Left Sidebar
- 知识分类导航:
  - 全部知识条目
  - 需求文档(按业务模块分类)
  - 架构方案(按技术栈分类)
  - 代码模式(按语言/框架分类)
  - 问题解决方案(按问题类型分类)
  - 部署配置(按环境分类)
  - 我的收藏
- 快速筛选器:
  - 最近30天
  - 高频引用
  - AI推荐
  - 待完善标记

## Main Content Area
- 页面标题区:
  - "知识库"标题
  - 统计信息:总条目数、本月新增、引用次数
  - 搜索框:
    - 全文搜索功能
    - 智能搜索建议(AI辅助)
    - 高级筛选展开按钮
  - 视图切换:卡片视图(当前选中) / 列表视图
  - 排序选项:最新创建 / 引用次数 / 相关度
- 高级筛选面板(展开时显示):
  - 时间范围选择器
  - 知识类型多选:需求/架构/代码/问题/部署
  - 技术标签多选:前端/后端/数据库/DevOps
  - 创建者筛选
  - 质量评分筛选
  - 应用筛选按钮
- 知识条目卡片列表:
  - 知识卡片(多个):
    - 条目类型图标和标签
    - 条目标题
    - 摘要描述(前150字)
    - 技术标签:React / Node.js / PostgreSQL 等
    - 引用次数统计
    - 质量评分(由AI和人工评价综合计算)
    - 创建时间和来源(关联的需求或任务ID)
    - 快速操作:
      - 查看详情按钮
      - 收藏/取消收藏按钮
      - 引用到新需求按钮
      - 报告问题按钮(标记为过时或有误)
  - AI推荐卡片(穿插在列表中):
    - "相似知识推荐"标识
    - 基于当前浏览的推荐条目
    - 关联理由说明
- 空状态提示(当搜索无结果时):
  - 插图
  - "未找到匹配的知识条目"提示
  - 调整搜索条件建议
  - 查看全部知识按钮
- 分页器:
  - 每页显示15条记录
  - 页码导航
  - 总记录数显示
