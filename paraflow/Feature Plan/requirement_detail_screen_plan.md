# 需求详情
展示单个需求的完整信息,包括用户原始输入、AI生成的结构化需求文档、任务拆解结果,以及整个需求的执行历史记录,支持编辑需求和提交架构审核。

Layout Hierarchy:
- Header (Full-width):
  - Top Navigation Bar
- Content Container (Positioned below the header):
  - Main Content Area with Two-Column Layout

## Top Navigation Bar
- 产品Logo和名称
- 导航菜单:工作流看板、需求管理、知识库、系统设置
- 全局搜索框
- 新建需求按钮
- 通知中心图标
- 用户头像和下拉菜单

## Main Content Area
- 面包屑导航:
  - 需求管理 > 需求详情 > [需求ID]
- 需求概览卡片:
  - 需求ID和标题(可编辑)
  - 当前状态徽章
  - 优先级选择器
  - 创建者和创建时间
  - 最后更新时间
  - 操作按钮:编辑、归档、删除
- 左侧内容区(主要信息):
  - 原始需求描述:
    - 用户输入的需求描述文本(可编辑)
    - 附件列表(如有)
  - AI生成的需求文档:
    - 结构化需求文档内容展示
    - 包含:业务目标、用户场景、功能范围、验收标准
    - AI分析时间戳
    - 重新分析按钮
  - AI提出的澄清问题(如有):
    - 问题列表
    - 用户回答记录
    - 添加补充说明按钮
  - 技术架构方案(如已生成):
    - 技术选型说明
    - 模块划分图示
    - 接口设计概要
    - AI架构师生成时间戳
    - 提交审核按钮(如状态为待审核)
  - 任务拆解列表(如已审核通过):
    - 任务卡片列表:
      - 任务ID和标题
      - 任务类型:前端/后端/测试/部署
      - 预计工时
      - 当前状态和负责AI Agent
      - 进入任务按钮
- 右侧侧边栏(辅助信息):
  - 时间线:
    - 需求创建
    - AI分析完成
    - 用户补充说明
    - 架构方案生成
    - 审核节点记录
    - 任务分配
    - 开发进展里程碑
  - 相关需求推荐:
    - 相似历史需求链接(基于AI匹配)
    - 快速参考按钮
  - 统计信息:
    - 关联任务总数
    - 已完成任务数
    - 预计总工时
    - 实际消耗工时
