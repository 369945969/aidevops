# 工作流看板
一个实时展示所有开发任务状态的可视化看板页面,采用看板式布局,按照需求分析、架构设计、开发中、代码审查、测试、待审核、已完成等阶段分列显示任务卡片,支持快速定位需要人工审核的关键节点。

Layout Hierarchy:
- Header (Full-width):
  - Top Navigation Bar
- Content Container (Positioned below the header):
  - Main Content Area

## Top Navigation Bar
- 产品Logo和名称
- 导航菜单:工作流看板(当前选中)、需求管理、知识库、系统设置
- 全局搜索框
- 新建需求按钮
- 通知中心图标(显示待审核数量)
- 用户头像和下拉菜单

## Main Content Area
- 页面标题区:
  - "工作流看板"标题
  - 筛选器:全部任务 / 我的审核 / 今日更新
  - 视图切换:看板视图(当前选中) / 列表视图
  - 时间范围选择器
- 看板状态列:
  - 需求分析列(2个任务卡片)
  - 架构设计列(1个任务卡片,待审核状态高亮)
  - 开发中列(4个任务卡片,显示AI Agent并行工作)
  - 代码审查列(2个任务卡片)
  - 测试列(1个任务卡片)
  - 待审核列(3个任务卡片,醒目提示)
  - 已完成列(5个任务卡片)
- 任务卡片内容:
  - 需求ID和标题
  - 当前负责的AI Agent类型图标和名称
  - 任务优先级标签
  - 预计完成时间倒计时
  - 进度指示器
  - 快速操作按钮:查看详情、审核(如需要)
- 实时更新指示:
  - 页面右下角显示"实时同步中"状态
  - 新增或状态变更的任务卡片有动画高亮提示
