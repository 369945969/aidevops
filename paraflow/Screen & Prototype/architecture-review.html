REQ-0008 多语言支持模块
查看完整需求

系统需要支持中英文双语界面切换，需要设计可扩展的国际化架构，支持动态语言包加载和翻译管理。

AI架构方案详情

技术选型

前端: React + i18next

选择i18next作为国际化框架,支持JSON语言包加载、插值翻译和惰性加载

后端: Node.js + 翻译管理API

提供翻译管理后台API,支持语言包CRUD和版本管理

数据库: PostgreSQL + JSONB

使用JSONB字段存储翻译内容,支持按语言高效查询和更新

AI决策依据: 参考项目REQ-0003技术栈,保持一致性

数据存储模块 PostgreSQL JSONB翻译存储

接口设计

GET /api/locales/{lang} 获取指定语言的翻译包
POST /api/locales 创建新语言翻译包
PUT /api/locales/{lang}/keys/{key} 更新指定翻译键值
GET /api/locales/list 获取支持的语言列表
